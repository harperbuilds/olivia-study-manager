
# VISION
학생은 공부에만 집중하고, 기록은 시스템이 책임진다

# Mission
최소한의 입력으로 학생의 학습 과정을 자동으로 기록하여,
학생이 기록이 아닌 공부와 성장에 집중할 수 있는 환경을 만든다.

# Core Values

## 1. Less Input
학생의 입력을 최소화한다. 

## 2. Study First
공부보다 기록에 시간을 쓰지 않는다. 

## 3. Groth
자동으로 기록된 학습 데이터를 통해 
학생이 자신의 성장을 체감할 수 있도록 한다. 

## 4. Automation
학생이 해야 할 반복적인 기록은 시스템이 자동으로 수행한다.
